# Importação de planilhas

Use os templates na raiz do projeto:

- `template-importacao.csv`
- `template-importacao.xlsx`

No app: **Importar → Planilha CSV ou Excel** (mês selecionado no calendário).

Colunas: **Descrição**, **Categoria**, **Tipo** (Entrada / Despesa / Investimento), **Valor**, **Status**, **Tag**, **Observação**.

Tags: Gabriel, Barbara, Casa, Família.

O template inclui lançamentos de exemplo (Jun/2026) — edite ou apague antes de importar seus dados reais.
